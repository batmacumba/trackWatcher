## Dependencies
* python 3
* pip

## Use pip to install the following packages
```console
foo@bar:~$ pip install pyautogui pydirectinput pywin32 watchdog
```

## Configuration:
* Make sure the track record keybind in the game is set to LCTRL-R

## Run the script:
```console
foo@bar:~$ python trackWatcher.py
```
